Project on Console

Visual Studio Code is an editor for writing html,css and javascript code and making web apps. It can be downloaded and installed on the computer. Try it out!!

Console screen is helpful in testing out the javascript code. You can open it by right clicking on the page. You go to inspect page and go to console tab. To print anything on console you have to go to the console page, and use the inbuilt function "console.log()" type anything in the brackets and put quotes.There you have it when you hit enter!
